What's New in AtlasDays

New features and improvements in each release.

Version 0.6.2 — March 12, 2026

New

  • Future trips now appear in a dedicated Upcoming section in the trip list
  • Country search recognises common alternative names — search "Turkey", "USA", "UK", "Holland", and more

Changed

  • Add Trip redesigned as a half-sheet with a compact header and popup date pickers
  • Airport Transit and Notes tucked behind a More options row to keep the form clean
  • Various form label and layout improvements

Fixed

  • Future trips no longer counted in the Travel Summary
  • Ongoing trips now count only days up to today in the Travel Summary

Version 0.6.1 — March 12, 2026

Changed

  • Various layout and polish improvements to the country detail sheet and full-screen map
  • Notes field in Add/Edit Trip starts as a single line and grows as you type

Fixed

  • Trips added from the country detail sheet now appear immediately
  • Transit badge was invisible in dark mode

Version 0.6.0 — March 11, 2026

New

  • Settings tab — a dedicated screen for all your preferences, replacing the World Map tab
  • Toggle auto-detect country changes and push notifications directly from Settings
  • Choose your home country in Settings (with a "No fixed residence" option)
  • iCloud Sync toggle — turn iCloud backup on or off from Settings
  • Set a default trip mode (Exact or Approximate) for new trips
  • Theme picker — switch between System, Light, and Dark appearance
  • CSV export, import, and template download from Settings
  • Density toggle — switch between comfortable and compact trip list layout
  • Re-run onboarding at any time from Settings

Changed

  • The world map is now accessible by tapping the minimap on the Summary tab
  • Menus on the Trips and Summary tabs have been simplified

Fixed

  • Notification toggle no longer gets stuck after granting permission
  • Home country selection now saves reliably

Version 0.5.1 — March 11, 2026

New

  • Re-run onboarding from the Summary tab menu
  • "No fixed residence" option during onboarding
  • Optional residency tracker when selecting a home country

Changed

  • Onboarding redesigned — simpler country selection with an optional residency tracker

Fixed

  • Location detection no longer suggests your home country right after onboarding

Version 0.5.0 — March 9, 2026

New

  • Country detail sheet — tap any country on the world map to see your trip history, trip count, and total days spent
  • Add Trip button directly from the country detail sheet
  • Full-screen map by tapping the minimap on the Summary tab
  • Map color style toggle — choose between blue or greyscale themes

Changed

  • Transit countries (airside) are no longer colored on the map

Version 0.4.0 — March 8, 2026

New

  • New tracker counting mode: Per Stay (alongside Rolling and Calendar Year)
  • Optional "Tracking starts on" date to ignore days before a chosen date
  • Country flags in the country picker
  • Splash screen with branded logo animation on startup

Changed

  • Tracker editor reorganized into clearer sections
  • Various performance improvements — faster startup and smoother scrolling

Version 0.3.0 — March 5, 2026

New

  • iCloud sync — trip data is automatically backed up to your personal iCloud
  • Undo banner when deleting a trip — tap within 4 seconds to restore it
  • Delete Trip button in the trip edit form

Changed

  • Date precision simplified to two options: Exact and Approximate
  • Exact date trips can now have future dates — useful for planned trips
  • Improved CSV format with flexible month input

Version 0.2.0 — February 28, 2026

New

  • Trip list with sections by year, Ongoing, and No Date
  • Auto-detection — the app suggests starting or ending a trip when you enter a new country
  • Summary tab with visa day totals, custom date ranges, and rolling window presets
  • Visa rule trackers (e.g. Schengen 90/180) with progress bar and days remaining
  • Interactive world map showing visited countries
  • CSV export and import

Version 0.1.0 — December 22, 2025

  • Initial release — basic trip logging with country, dates, and notes
