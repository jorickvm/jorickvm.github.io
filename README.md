# jorickvm.github.io

Marketing site for [AtlasDays](https://atlasdays.app), a private iPhone app for tracking trips by country, visa-day limits, tax residency thresholds, and travel history.

Hosted via GitHub Pages at `atlasdays.app`.

## Structure

```
index.html       Main marketing page
privacy.html     Privacy policy   (/privacy)
changelog.html   Release notes    (/changelog)
assets/          Images (logo, favicon, screenshots)
```

## Local preview

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
