# jorickvm.github.io

Marketing site for [AtlasDays](https://jorickvm.github.io/atlasdays.html), a private iPhone app for tracking trips by country, visa-day limits, tax residency thresholds, and travel history.

Hosted via GitHub Pages at `jorickvm.github.io`.

## Structure

```
atlasdays.html            Main marketing page
atlasdays-privacy.html    Privacy policy
atlasdays-changelog.html  Release notes
assets/                   Images (logo, favicon, screenshots)
```

## Local preview

Open `atlasdays.html` in a browser, or run a local server:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000/atlasdays.html`.
