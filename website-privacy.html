AtlasDays Website Waitlist Notice

Last updated: April 16, 2026

AtlasDays is now live on the App Store. This page applies only to historical email addresses submitted through the retired website launch waitlist before release. It does not describe the in-app privacy model.

Key Points

The website launch waitlist is closed and no longer collects new email addresses.

This page covers only historical waitlist emails submitted before the App Store release.

You can contact us to ask for a historical waitlist email to be removed.

1. Current Status

The launch waitlist form previously used on the AtlasDays website has been retired. AtlasDays is now available on the App Store at the live App Store listing.

2. Historical Data Covered by This Page

If you submitted the old launch waitlist form before release, the website collected the email address you provided so launch-related updates could be sent.

3. Historical Form Processing

The retired launch waitlist form was processed through Formspree as the website form-processing service.

4. Scope

  • This page applies only to the retired website launch waitlist.
  • The waitlist was always separate from the AtlasDays app itself.
  • The form collected only an email address.
  • AtlasDays does not sell waitlist data.

5. Removal Requests

If you previously joined the website waitlist and want that email removed, contact support email.

6. Current Privacy Documents

For the current app privacy model, read the AtlasDays App Privacy page. For product behavior and settings, start in the Help Center.
