AtlasDays Website Privacy

Last updated: April 16, 2026

This page covers the public AtlasDays website. It does not describe the in-app privacy model.

Key Points

The website uses analytics to understand visits, pages viewed, and traffic sources.

The old website launch waitlist is closed and no longer collects new email addresses.

You can contact us to ask for a historical waitlist email to be removed.

1. Website Analytics

The website uses Google Analytics to measure website traffic and understand which pages and sources are useful. This may include pages viewed, approximate location, device and browser information, referring websites, search engines, and campaign parameters in links. Google explains how it uses information from sites that use its services at google.com/policies/privacy/partners.

This is website measurement only. It does not add analytics tracking inside the AtlasDays app.

2. Historical Waitlist Data

The launch waitlist form previously used on the AtlasDays website has been retired. AtlasDays is now available on the App Store at the live App Store listing.

If you submitted the old launch waitlist form before release, the website collected the email address you provided so launch-related updates could be sent.

3. Historical Form Processing

The retired launch waitlist form was processed through Formspree as the website form-processing service.

4. Scope

  • This page applies to the public AtlasDays website.
  • The website and historical waitlist are separate from the AtlasDays app itself.
  • The retired waitlist form collected only an email address.
  • AtlasDays does not sell waitlist data.

5. Removal Requests

If you previously joined the website waitlist and want that email removed, contact support email.

6. Current Privacy Documents

For the current app privacy model, read the AtlasDays App Privacy page. For product behavior and settings, start in the Help Center.
